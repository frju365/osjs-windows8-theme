# OS.js v3 Windows 8 Style Theme

A style theme that imitates the Windows 8 look

[![Build Status](https://travis-ci.org/os-js/osjs-windows8-theme.svg?branch=master)](https://travis-ci.org/os-js/osjs-standard-theme)

![Screenshot](https://raw.githubusercontent.com/os-js/osjs-windows8-theme/master/screenshot.png)

## Installation

```bash
npm install --save --production @osjs/windows8-theme
npm run package:discover
```
